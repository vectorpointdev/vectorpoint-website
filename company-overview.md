# Vector Point — Company Overview

Vector Point (vector.dev LLC) is a solo software engineering practice founded by Andy, specializing in full-stack development, mobile apps, and AI-powered products. The practice partners with startups, founders, and growing teams who need high-quality engineering without the overhead of a large agency.

## Services

- **Full-Stack Development** — Web applications, REST & GraphQL APIs, databases, and cloud infrastructure. Core stack: Next.js, React, Node.js, PostgreSQL.
- **Mobile Development** — Native iOS with SwiftUI and cross-platform with React Native, from prototype through App Store launch.
- **AI Integration** — LLM-powered features, intelligent chatbots, workflow automation, and RAG pipelines using Claude, OpenAI, and custom models.
- **Freelance Engineering** — Embedding directly on client teams to ship their backlog with fast ramp-up, clean code, and on-time delivery.

## Notable Work

- **Don** (don.financial) — An AI-powered personal finance app featuring conversational interaction with financial data, spending tracking, and intelligent insights. Built with Next.js, SwiftUI, PostgreSQL, Claude API, and Plaid.
- **Fintech Dashboard** — A real-time analytics dashboard for a fintech startup, built from zero to production in 6 weeks using React, Node.js, WebSockets, and D3.js.
- **AI Workflow Tool** — An internal document review automation tool powered by LLMs that reduced manual review time by 80%, built with Python, FastAPI, React, and OpenAI with RAG.

## Technology

TypeScript, React, Next.js, Swift, SwiftUI, React Native, Node.js, Python, PostgreSQL, Prisma, Claude API, OpenAI, AWS, Docker, Vercel.

One engineer who owns the full stack — from database to design system — with a focus on clean code, fast iteration, and shipping real products to production.
